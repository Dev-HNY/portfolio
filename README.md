## üöÄ Local Development

1. Clone the repository:
```bash
git clone https://github.com/Dev-HNY/portfolio.git
cd portfolio
```

2. Open in browser:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve

# Or simply open index.html in your browser
```

3. View at `http://localhost:8000`
